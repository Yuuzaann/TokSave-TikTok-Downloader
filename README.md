# TokSave-TikTok-Downloader

TokSave is a modern web-based TikTok downloader that allows users to download videos without watermark, extract MP3 audio, and save TikTok photo slideshows easily.

Built using:
- HTML
- Bootstrap
- TailwindCSS
- Vanilla JavaScript

Features:
- Download TikTok videos without watermark
- Extract MP3 audio from TikTok
- Download TikTok photo slideshow
- Modern responsive UI
- Mobile friendly
